Quem sou eu

Meu nome é Thiago Meneghello

Sou formado em Curso Superior de Tecnologia Mecatrônica Industrial e atualmente desenvolvo projetos pessoais em Ciência de Dados para adquirir experiência na solução de problemas de negócio e domínio sobre ferramentas de Análise de Dados.

Tenho experiencia de 10 anos em manutenção industrial atuando em indústrias multinacionais na programação, manutenção e controle de processos automatizados, programação de células robóticas, para garantir a confiabilidade e qualidade de produção.

Busco uma oportunidade para trabalhar profissionalmente como Cientista de Dados para melhorar a tomada de decisão da empresa, através da construção de soluções utilizando dados.

Projetos em Ciência de Dados

Desenvolvimento de um Plano Gerencial para Negócio com o Streamlit

Para esse projeto foi utilizado conceitos de Programação em Python para manipulação e visualização de dados, raciocínio analítico e estratégico em lógica de negócios e ferramentas web como Streamlit e GitHub. Ao final, apresento um painel gerencial com gráficos e tabelas (dashboard) com as principais métricas de uma empresa marketplace de restaurantes com atuação global.

O resultado final do projeto foi hospedado em um ambiente Cloud e disponibilizado através de um link web. O painel gerencial pode ser acessado por qualquer dispositivo conectado à internet.

  • Python para manipulação de dados

  • Jupyter Lab e Terminal
  • Streamlit e Streamlit Cloud
